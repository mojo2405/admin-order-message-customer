== Description ==

Woocommerce Admin Order Message Customer will let you text your customer via Whatsapp without saving them as contacts on your phone's contacts library.

Major features in this plugin include:

* Directly open Whatsapp application with the customer phone number taken from the billing phone field
* No need to save the contact phone number on your phone

== Installation ==

Upload the plugin to your Woocommerce website, activate it, and that's it.
